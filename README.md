1) Run VENV library module as script 
   * `python -m venv venv`
2) Activate VENV
    * `venv/Scripts/activate`
3) Install aiogram
    * `pip install aiogram`
    * `pip install aiogram_calendar`
4) Install xampp
    * start *Apache* and *MySql* module

    * press *Admin* button in *MySql* module
   ![img_2.png](img_2.png)
    * press **New** button
   ![img_3.png](img_3.png)
    * fill db name with **tour_bot** and create
